Bridging Education and Regional Languages using AI

## Project Overview
EduSpeak is an AI-powered application designed to make educational content accessible to regional language learners.
It extracts content from PDFs, summarizes it, translates it into a regional language (Tamil), and converts the output into speech.

## Features
- PDF Content Extraction
- AI-based Text Summarization
- English to Tamil Translation
- Text-to-Speech Conversion
- Simple Web Interface

## Tech Stack
- Python
- Streamlit
- NLP (Summarization)
- Translation APIs
- Text-to-Speech (TTS)
